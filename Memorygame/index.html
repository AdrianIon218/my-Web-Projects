The memory game is a common children's game played with a set of cards.

Each card contains a picture on one side that matches the same picture from another card.

The game starts with all the cards face down .
The player must click a card in order to reveal its image. Afterwards the player must click another card that he/she thinks matches the image of the previous card .

If the two cards have the same picture, then they dissapear, otherwise they turn face down again.

The player will win the game when all the cards have been matched.
